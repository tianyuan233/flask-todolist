# fuckingflask_todolist

基于flask和mysql的一个小项目

包括了 用户注册，登陆，邮箱验证

添加todolist，显示过期项目

## 项目下载及运行:

- 基础运行环境安装:

```
pip install -r reqquirements.txt
```
- mysql文件为mytodolist.sql
 
```
git clone https://github.com/tianyuan233/fuckingflask_todolist
cd fuckingflask_todolist
python manage.py runserver
```
